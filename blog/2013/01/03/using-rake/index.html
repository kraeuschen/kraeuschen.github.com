Using Rake

Jan 3rd, 2013

You can use rake for creating sites and posts. This file ist adopted from jekyll-boostrap.

Starting server

rake preview

Create page

rake page name="about.md"

or

rake page name="about/index.html"

Create a post

rake post title="A Title" [date="2012-02-09"]
